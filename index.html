Bustub

github: https://github.com/PinkDoc/bustub-2020

CMU15-445 基于 C++ 开发的支持简单 SQL 操作的单机数据库： 底层基于 LRU 设计了 BufferPool，所有 page 操作在 BufferPool 上进行。同时在 Buffer Pool 实现了B+树，使用LatchCoupling算法进行并发控制。 语句执行采用火山模型，支持 SELECT，INSERT, DELETE，UPDATE，JOIN，AGGREGATION，LIMIT操作。Join 有 Nested Loop Join 和 Index Join 两种实现。 并发控制采用 2PL 设计，实现 RU, RC, RR 3种隔离级别。 实现全局的 Lock Manager 管理 R/W 锁，死锁处理是基于死锁预防中的 wound-wait 算法。

项目难点： 实现基于磁盘并发的B+树 实现并发控制的LatchCoupling算法。
